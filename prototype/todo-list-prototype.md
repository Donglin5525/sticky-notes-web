## 二、四象限任务管理

### 2.1 四象限区域
| 象限 | 名称 | 描述 | 颜色标识 |
|------|------|------|----------|
| 第一象限 | 优先事项 | 紧急且重要，需立即处理 | 红色 |
| 第二象限 | 战略项目 | 重要不紧急，需规划时间 | 蓝色 |
| 第三象限 | 琐碎事务 | 紧急不重要，可委托他人 | 黄色 |
| 第四象限 | 陷阱区域 | 不紧急不重要，尽量避免 | 灰色 |

### 2.2 单个任务卡片
```
┌─────────────────────────────────────┐
│ ○ 完成产品需求文档                   │
│   预计时间: 2小时  ⏰ 10:00-12:00    │
│   #工作 #产品                        │
│                          [✓] [✎] [🗑]│
└─────────────────────────────────────┘
```
- 圆形复选框：标记完成状态
- 任务标题
- 预计时间和时间段（可选）
- 标签
- 操作按钮：完成、编辑、删除

## 六、数据库设计预览

| 表名 | 说明 |
|------|------|
| daily_tasks | 每日任务记录 |
| daily_summaries | 每日总结记录 |
| prompt_templates | Prompt 模板 |
| ai_analysis_history | AI 分析历史 |

## 七、交互流程

1. **Tab 切换**: 点击顶部 Tab 在「便签笔记」和「待办清单」之间切换
2. **添加任务**: 点击「+ 添加任务」按钮，弹出添加任务弹窗
3. **完成任务**: 点击任务前的复选框标记完成
4. **拖拽任务**: 支持将任务拖拽到不同象限
5. **今日总结**: 在底部区域填写今日收获、反思和计划
6. **智能分析**: 点击「智能总结」按钮，选择模板生成分析
7. **查看历史**: 通过日历选择日期查看历史记录
8. **数据看板**: 切换年/月/日维度查看统计数据
