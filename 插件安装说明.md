# Sticky Zen - Chrome 插件安装指南

## 1. 解压文件
您已经下载了 `sticky-notes-extension.zip` 文件。请将其解压到一个文件夹中（例如 `sticky-notes-extension`）。

## 2. 打开 Chrome 扩展程序管理页面
1.  在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
2.  或者点击浏览器右上角的三个点图标 -> **扩展程序** -> **管理扩展程序**。

## 3. 开启开发者模式
在扩展程序页面的右上角，找到 **"开发者模式"** 开关，并将其打开。

## 4. 加载已解压的扩展程序
1.  点击左上角的 **"加载已解压的扩展程序"** 按钮。
2.  在弹出的文件选择窗口中，选择您刚才解压的文件夹（确保选择了包含 `manifest.json` 的那个文件夹）。

## 5. 开始使用
*   **点击图标**：在浏览器右上角的扩展程序栏中，点击 Sticky Zen 图标即可打开便签。
*   **快捷键**：默认快捷键为 `Alt+S`（Mac 上为 `Option+S`）。您可以在 `chrome://extensions/shortcuts` 中自定义快捷键。

## 常见问题
*   **数据丢失？** 插件的数据存储在浏览器的本地存储中。如果您卸载插件或清除浏览器数据，便签可能会丢失。建议定期使用插件内的 "Backup" 功能导出备份。
*   **界面显示不全？** 插件默认窗口大小为 800x600。如果显示异常，请尝试调整浏览器缩放比例。
