# Sticky Zen - Chrome 插件安装指南（修复版）

## 1. 解压文件
您已经下载了 `sticky-notes-extension-fixed.zip` 文件。请将其解压到一个文件夹中。

**⚠️ 关键步骤：**
解压后，请打开文件夹确认一下，您应该能**直接看到** `manifest.json` 文件。
如果看到的是另一个文件夹（例如 `public` 或 `dist`），请进入那个文件夹，直到看到 `manifest.json` 为止。

## 2. 打开 Chrome 扩展程序管理页面
1.  在 Chrome 浏览器地址栏输入 `chrome://extensions/` 并回车。
2.  或者点击浏览器右上角的三个点图标 -> **扩展程序** -> **管理扩展程序**。

## 3. 开启开发者模式
在扩展程序页面的右上角，找到 **"开发者模式"** 开关，并将其打开。

## 4. 加载已解压的扩展程序
1.  点击左上角的 **"加载已解压的扩展程序"** 按钮。
2.  在弹出的文件选择窗口中，**选择包含 `manifest.json` 文件的那个文件夹**。

## 5. 成功！
现在您应该能看到 "Sticky Zen" 插件已成功加载。点击浏览器右上角的插件图标，或按 `Alt+S` 即可使用。
